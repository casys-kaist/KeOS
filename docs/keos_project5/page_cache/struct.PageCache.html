Struct PageCache

Source 
pub struct PageCache<FS: FileSystem>(pub Arc<PageCacheInner<FS>>);
Expand description

A reference-counted handle to the page cache.

PageCache wraps an [Arc] around PageCacheInner, allowing safe sharing across multiple threads. It provides methods to construct a cache and to read pages with read-ahead support.
