Expand description

Represents a kernel virtual address.

The Kva (Kernel Virtual Address) struct is a lightweight wrapper around a usize value that represents an address in the kernelโs virtual address space. It provides utility methods for address validation, conversion, and alignment.

This abstraction ensures that kernel addresses are used safely and consistently, reducing the risk of incorrect memory access.
