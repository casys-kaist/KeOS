Struct PageTableRoot

#[repr(align(4096))]
pub struct PageTableRoot([Pml4e; 512]);
Expand description

A page table root.

It wraps the Pml4e array to ensure page table align to 4096. Note that it is not allowed to modified the larger indices than Self::KBASE, which are reserved for kernel address.
