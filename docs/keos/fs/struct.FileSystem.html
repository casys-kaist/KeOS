Expand description

A global file system abstraction.

The FileSystem struct provides an interface for interacting with the file system, including operations such as opening and creating files.

§Example

let fs = file_system();
if let Some(file) = fs.open("example.txt") {
    println!("Opened file: {}", file.name());
}
